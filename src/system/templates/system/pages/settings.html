{% if tab == 'telegram' %}

Зачем: Telegram-бот нужен для мгновенных уведомлений о новых заказах и для восстановления пароля.

Пошаговая настройка:

  1. Откройте Telegram, найдите @BotFather и создайте нового бота. Скопируйте полученный Token.
  2. Вставьте Token в поле «Bot Token» на этой странице.
  3. Откройте диалог с вашим новым ботом в Telegram и нажмите «Запустить» (или отправьте /start).
  4. Вернитесь сюда и нажмите «Get Chat ID» — система найдёт ваш диалог автоматически.
  5. Нажмите «Сохранить», затем «Тестовое сообщение» для проверки связи.

Важно: Между отправкой /start и нажатием «Get Chat ID» должно пройти не более 15 минут.

{% elif tab == 'security' %}
  • Смена пароля: Введите новый пароль (минимум 5 символов) и подтвердите его. Текущий пароль не требуется — доступ уже подтверждён сессией.
  • Хранение: Пароли хранятся в зашифрованном виде. Исходный пароль восстановить из базы невозможно.
  • Восстановление через Telegram: Если вы забыли пароль, на странице входа можно получить одноразовый код в Telegram и войти без пароля. После входа смените пароль здесь.
  • ⚠ Важно: Тот, у кого есть доступ к вашему Telegram-аккаунту, может войти в панель без пароля. Защитите Telegram двухфакторной аутентификацией.
